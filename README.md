## Development
### Setup
- Install nodejs version 8 <
- Install gitbook
    ```
    npm i -g gitbook-cli
    ```
- run the gitbook
    ```
    gitbook serve
    ```

### Menambah Materi
Untuk membuat halaman baru, buat file markdown (.md) baru, lalu daftarkan halaman tersebut ke `SUMARRY.md` dengan format `* [Nama Halaman](nama_file.md)`
